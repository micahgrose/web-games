the myth

"A bigger context window always makes it smarter."

✗ The myth: more memory = a better model.
✓ The truth: long context only helps if the task needs it. These replies depend mostly on your last turn, so extra memory has nothing to do — and it makes every step slower.
👀 Watch: start at memory 128. Train — auto-pauses at 1500 steps. Notice how slowly it crawls there (Speed even drops to fractional), and where generalized lands.
🔁 Drop memory (the only unlocked dial) to 16, then 64. Each change resets, so Train to 1500 each time. It reaches 1500 much faster, and generalized is about the same. The bigger window bought nothing but slowness.
