what each setting does

speed · capacity · quality

Speed — steps per second. Pace, not smarts.

memory — context window. More history, slower; wasted on short chats.

neurons — width / capacity. More ≠ smarter: on little data it overfits.

data — the knob where "more" usually helps, if it's varied.

creativity / anti-repeat — chat-time: randomness, and how hard it avoids repeating itself.
