about this chapter

Chapter 2 — a network learns to write

What this is: a character-level network that learns to greet, typed one letter at a time, each predicted from the letters before it. Some training greetings are misspelled on purpose, so it learns to read sloppy input.

Step up from TinyBrain: TinyBrain gave one answer (a colour); this produces a sequence — language. It also adds the dashboard: memorized vs generalized vs confidence.

The limit: it only knows greetings and reads letters, not meaning. Chapter 3 fixes that with whole words and attention.

This demo's settings: neurons 64 · data 60.
Why these? enough width to learn the greeting patterns, enough varied data to generalize — without tipping into overfitting. (Change them in free play.)
