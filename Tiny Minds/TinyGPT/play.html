what each setting does

speed · capacity · quality

Speed — steps per second. Pace, not smarts.

memory — context window. More history, slower (attention cost grows with its square); wasted on short chats.

neurons — width / capacity. More ≠ smarter: on small data it overfits.

heads — parallel attention spotlights; more relationships at once, more compute.

layers — stacked stages of understanding; deeper, but slower and needs more training.

data — the one knob where "more" usually helps, if it's varied.

creativity / anti-repeat — chat-time: randomness, and how hard it avoids repeating itself.
