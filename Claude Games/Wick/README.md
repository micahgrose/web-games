# wick

keep your flame. outlast the dark.

a survival game about a candle that refuses to go out.
open `index.html` in a browser. wasd to move. that's all you need to know.

## the night (v2)

- **choose your first flame** — start as any of five weapons; each plays differently.
- **evolutions** — max a flame and pair it with the right gift, and it becomes something greater:
  Sunpiercer, Halo of Ash, Pyre, Wildfire, Wisp Pack. claim it the moment it's offered.
- **new threats** — gloom (armored tank), wisp (drinks your light), thief (steals your sparks and runs).
- **two boss walls before the dawn** — hush, the gaunt lantern (a spiral-caster that summons
  light-eaters) at ~4:45 and ~9:20, then mourn, the unlit (a three-phase charger) at first light.
- the night runs about **twelve minutes**. endless mode after, alternating both bosses.

built with plain canvas + web audio. no libraries, no build step.
